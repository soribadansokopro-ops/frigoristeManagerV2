# FRIGORISTE MANAGER - GAME VISION

## Positionnement
Frigoriste Manager est un jeu video de simulation et progression.
Ce n est ni un SaaS, ni un dashboard metier, ni un logiciel de supervision.

## North Star
Chaque fonctionnalite doit rendre le jeu:
- plus amusant
- plus immersif
- plus satisfaisant
- plus rejouable

Question de validation produit:
"Est-ce que cela donne envie de continuer a jouer ?"

## Piliers de gameplay
1. Boucle terrain (court terme)
- Recevoir mission
- Aller sur site
- Observer
- Mesurer
- Diagnostiquer
- Reparer
- Valider
- Recompense

2. Boucle progression (moyen terme)
- XP
- Argent
- Deblocage outils
- Deblocage niveaux
- Difficultes croissantes

3. Boucle meta (long terme)
- Camion et atelier
- Recrutement employes
- Gestion entreprise
- Contrats premium
- Reputation

## Direction UX/UI
- HUD de jeu moderne, lisible, vivant
- Animations utiles et feedbacks instantanes
- Ecrans specialises (une page = un objectif clair)
- Recompenses visuelles et sonores
- Forte coherence visuelle entre toutes les scenes

## Regles de conception
- Gameplay avant realisme absolu
- Realisme au service du fun
- Choix du joueur a chaque etape importante
- Tutorisation contextuelle, jamais envahissante
- Lisibilite prioritaire en action

## Structure fonctionnelle cible
1. Home Hub (garage / entreprise)
- Progression, finances, flotte, equipe

2. Missions Board
- Contrats classes par niveau, gains, risque, duree

3. Intervention Mode
- Deplacement site
- Analyse panne
- Sequence mesures / diagnostic / reparation

4. Post-Mission
- Debrief score, erreurs, bonus, apprentissages

## Success Criteria
- Session courte satisfaisante en moins de 10 minutes
- Sensation de progression a chaque mission
- Systeme de debloquage motivant
- Interface percue comme jeu, jamais comme logiciel

## Decision Policy
Toute decision produit, design ou code doit respecter ce document.
Si une implementation ressemble a un logiciel metier, elle doit etre revue.
